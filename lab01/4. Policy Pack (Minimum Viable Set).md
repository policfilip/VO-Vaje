**Policy Pack (Minimum Viable Set)**

**1\. Access Control Policy**

\-Disable shared accounts / convert them into personal accounts  
\-Enable MFA for critical systems and services  
\-Setup DDoS protection  
\-Apply role-based access control  
\-Perform quarterly access review

**2\. Backup & Recovery Policy**

\-Setup quarterly backup testing  
\-Implement 3-2-1 backup strategy  
\-Store one backup copy offline or immutable

**3\. Incident Response Policy**

\-Setup escalation procedure  
\-Define incident response team roles  
\-Establish incident reporting procedure

**4\. Vendor Security Policy**

\-Require annual reviews  
\-Review availability for services / SLA  
\-Assess vendor security compliance (GDPR / ISO 27001\)

**5\. Acceptable Use Policy**

\-Restrict any USB keys / external storage  
\-Prohibit installation of unauthorized software  
\-Require device lock after inactivity
